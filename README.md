# [hits.link](https://hits.link) - the better hits counter

## Getting Started
```bash
# Copy all .env.example files into your .env file

# Run Docker Compose 
$ docker-compose up -d

$ cd api/prisma && go run github.com/prisma/prisma-client-go db push

# Start Go API
$ cd api && go run main.go

# Install dependencies
$ cd web && yarn

# Run web
$ cd web && yarn dev
```

## Stack
```bash
Frontend:
- Next.js
- TypeScript

Backend:
- Go

- Prisma
```
